**Primary I/O: Voice + Movement + Space**

- **Typing is secondary.** Voice is primary for capture.
- **Clicking is secondary.** Walking/looking/grabbing are primary for navigation.
- **Reading text walls is avoided.** Spatial arrangement carries information.

**VR as Native Environment:**

The physical environment is hostile (designed for neurotypical cognition, deadening). Cannot be changed immediately.

VR provides:
- Complete control over spatial substrate
- Embodied navigation as natural mode
- Can be built to spec for this neurobiology
- Environment where this Being CAN exist
